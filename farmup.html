FarmUP! Is an android application to help farmers to manage pest control schedule and keep track on all the pesticide record which is applied to each vegetable.
Problem Statement
There is a quite complicated and strict pesticide regulation in Japan. However most of farmers in Japan are managing pesticide and pest control manually, just write down everything on note. It could cause serious mistakes which can effect to our health. Also the pesticide regulation is frequently changed and it is hard to track the newest information all the time. The farmers are already busy with their daily work and they wish to reduce their work to concern about pesticide.
Solution
The main function is pest control management, fields and plantation management, schedule management. For pest control registration, the app checks whether the pesticide is applicable for the certain plantation, whether the number of uses of the pesticide is within the regulation. It calculates the optimum amount of pesticide depend on the plantation information. it tracks all the pesticide data that is applied for each plantation. Pesticide reglation database is periodically retrieved from the server and can always keep up with the latest pesticide information.
Design Principles
In order to make it as simple as possible and easy to understand, I designed 4 main functions on the top of the screen, schedule management, pest control management, farm management, and plantation management. I designed it possible for users to choose colors for each plantations, so that they can be visually manage and keep track the plantation and pest control based on that color. Since it was a project with few resources, I made a simple layout and flow to reduce the work of engineers.

FarmUP!は農家さんが作物を育てる際に使用する農薬の管理を行うことができるAndroidアプリです。
問題定義
現在日本では、各野菜に対する農薬の制限、トラッキングや管理、最適な農薬量の計算、散布スケジュール、これらの複雑な農薬の管理を各農家が個別に、ほとんどの場合ノートなどに記録をつけるなどして管理しています。 各農家の個別の管理なため、抜け漏れや計算ミスなどを防ぐことが難しい。農薬の使用ルールは頻繁に変わるため常に最新の情報を取り入れなければならない。農薬の管理には時間がかかる、本来の農業に専念したい、などの問題がありました。
ソリューション
外で使用することがメインになるため、オフラインでも使用できるようネイティブアプリを採用しました。 農園、作物、各作物に対する防除（農薬を作物に散布する作業）の管理を行うことができます。防除登録の際はデータベースにある農薬のデータと照合し、作物に散布可能か、散布回数は規定以内かを自動でチェックし、さらにその作物に最適な農薬の量を自動で計算できます。 農薬のデータベースは定期的にサーバから取得、常に最新の農薬情報を保持できます。
デザイン
なるべくシンプルでわかりやすくするため、全体スケジュール管理、防除管理、農園管理、作物管理という４つの軸でタブ分けしたデザインにしました。 作物ごとにユーザーが色を指定できるようにし、その色を基準に視覚的に管理できるようにしました。リソースが少ないプロジェクトだったので、エンジニアの負担を減らすためシンプルなデザイン、フローを心がけました。
