Bunbogu.log is a web application that is able to check and write review about stationeries.
This web application is designed as a MVP(minimum viable product) to extract relevant feedbacks from users.
On this prototype application we are targeting only innovators and early adopters who are enthusiastic about stationaries.
As most users are expected to be accessed from smartphones due to functions specialized only to write reviews and viewing, layout and user interface are designed for smartphones screen.

文房具レビューサイト、文房具ログのデザインをさせていただきました。
このウェブアプリケーションはMVP戦略に乗っ取ったプロトタイプアプリケーションで、最低限の機能のみを実装しています。
ターゲットとして設定したのは文房具に深い思い入れがあり、意見の交換やレビューをしたいと思っている人たちです。
レビューを書く、見ることだけに特化した機能のためほとんどのユーザはスマートフォンからのアクセスであることが予想されたため、レイアウト、ユーザインターフィエスはスマートフォン重視で設計しています。
