taoru.com EC
I've got an offer to renew taoru.com website which was originally made in 1980s. The main renewal purpose was to make the old website to modern responsive website to increase access number and gain frequent customers.
In the original website, “Order original towel” flow which is the main function of the website seemed only for professional use and was too complicated for normal users to proceed, even though they are the main target of the website. So I boldly removed most of the unnecessary process from the order flow and simplified it in just two ways to order original towels.
Also I designed the site structure SEO friendly, so it's become appearing on the first page of organic search with the keyword "泉州タオル　オリジナル作成".
In addition, I designed some paper materials for taoru.com.

taoru.comサイトのリニューアルをさせていただきました。
リニューアル目的は、今まで20年以上あるサイトを現代風に一新、アクセス数を伸ばし顧客を獲得することでした。 オリジナルタオルの注文のフローが素人目に分かりづらかったので、思い切って用途、仕様という２軸にしぼった動線を作り、初めてオリジナルタイルを注文するユーザにも分かりやすい配慮をしました。
また、SEOに考慮してキーワードを選定、オーガニック検索上位を目指して設計しました。現在『泉州タオル　オリジナル作成』のキーワードで１ページ目にヒットすることに成功しています。
