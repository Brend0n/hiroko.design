Reusable Design Foundation for Fast Launches

Project: Baracoda Design System
Device: iOS, Android app
Role: Lead Product Design
At Baracoda, an AI innovation and consulting company with a B2B2C model, our team constantly worked on new product concepts. Each project had unique goals, but shared similar structures and user flows. To improve speed and consistency, I initiated an internal project to build a design system that could serve as a flexible foundation for all future projects.
THE CHALLENGE

UX SOLUTION #1

A Scalable Design System

We created a modular design system that helped the team move from concept to MVP quickly while maintaining a unified visual language.
UX SOLUTION #2

Flexible Theming Engine

We developed a theming system that allows developers to adjust parameters like color, typography, and spacing to instantly create different visual styles for each project.
UX SOLUTION #3

Standardized Core Flows

We standardized recurring flows such as account creation, password recovery, and device connection, saving time and ensuring UX consistency across all products.
